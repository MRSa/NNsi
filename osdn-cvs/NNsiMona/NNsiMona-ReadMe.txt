"いつでもどれでも２ちゃんねる" -- 他２ちゃんねるブラウザと同期をとるコンジット開発(目標)

[現バージョンの制約]
・VFS（メモリカード）上に取得済みスレの差分取得のみを実行します。
   → 取得エラーが発生した場合、HotSyncログにそのスレ名とURLを出力します。
   → 取得済みログのすべてを差分取得、あるいはお気に入りスレのみ差分取得を
     選択可能です。
・ログの取得は、「２ちゃんねる」および「まちBBS」のみ可能です。


[開発環境の説明]
・CDK 4.03を利用しています。(CDK 4.03が必要です。)
  → http://www.palmos.com/dev/tech/conduits/ よりダウンロード可能です。
    (Windows版は、http://www.palmos.com/dev/tools/cdk/win/ です。)

・VisualC++ .NET Standard (Vc7)を使用して開発しています。

・CDKをC:\CDK403にインストールした環境で作成しています。
   → NNsiMona.DLLの出力先も、C:\CDK403\Common\Bin になっています。

[概略ロードマップ]
・Alpha   : とりあえず動くものをつくる
   → バージョン番号なし、パッケージの日付で識別
・Beta    : リファクタしてちゃんときれいにクラス分けする
   → バージョン番号なし？ パッケージの日付で識別
・Release : メンテナンス開始! (ソースコードのリリース開始予定)
   → バージョン場号付与。

[対応ロードマップ]
・新着確認 (各スレ差分取得)
   → 対応済み
・お気に入りのみ新着確認
   → 対応済み

・母艦の２ちゃんブラウザと同期 (Phase1)
   → Palmから母艦へログコピー(母艦の未読ポイント更新)

・母艦の２ちゃんブラウザと同期 (Phase2)
   → 母艦からPalmへログコピー(Palmの未読ポイント更新)

・母艦の２ちゃんブラウザと同期 (Phase3)
   → ログ同期(ログデータの更新と未読ポイント交換)

・母艦の２ちゃんブラウザと同期 (Phase4)
   → 対応２ちゃんブラウザの拡充
