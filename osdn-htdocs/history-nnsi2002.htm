NNsi 更新履歴 (2002年版)

  • Version 0.94 Revision003 : 29/Dec/2002
    • メッセージ内に入っている異常なNULL文字を無視するよう変更
    • レス数が1005を超えているスレが取得できても参照できなかった不具合に対処
  • Version 0.94 Revision002 : 25/Dec/2002
    • PalmOS4.0以降のデバイスでは、書き込み時に2chへ送信する時刻をUTC基準で計算するように変更する。(これでPalm本体の時刻がPHTでも大丈夫な、はず。)
    • スレ参照画面でしおり設定ボタンをタイトルの左隣に移動し、メッセージ移動矢印の間隔を広げる。
    • 新着メッセージ確認時に、ストップチェックを毎回行うようにした。
    • スレ番指定取得で、選択していないBBSを有効にするとBBSリストがおかしくなる不具合に対処したつもり。
  • Version 0.94 Revision001 : 20/Dec/2002
    • サポートWebブラウザにEudora Webを追加する
    • OS31NNsiで、スレ参照画面を開こうとすると無限ループに陥っていた不具合に対処
    • CLIE NX/NR用のシルク制御を行わないNNsi設定-4を追加
    • スレ一覧画面でBackボタンを押すと機能選択メニューが出るようになるNNsi設定-4を追加
  • Version 0.93 Revision002 : 14/Dec/2002
    • メッセージの解析に失敗するレスがある部分(レス番号が飛ぶ)について修正
    • レス番号(>>xxx)の解析部について、1000以上はレス番号と認識しないようにした
    • 差分取得に失敗すると、スレ状態がおかしくなり、以降参照できなくなる不具合に対処
    • グラフィック描画モード使用時に、スレの末尾にトーフが表示される不具合が発生する場合があることに対処
    • Jumpリスト表示で、ジョグPUSHがOKボタンを押したことにならない不具合に対処
  • Version 0.93 Revision001 : 12/Dec/2002
    • NNsi設定-3に、「新着確認」後に未読一覧へ移動する設定を用意した
    • ギコのしっぽで作成されたログで、スレ参照番号が ">"にならず"&gt;"と表示されていた不具合に(一応)対処した
    • スレ内ジャンプ機能の強化。レス番号を入力せず "Go" ボタンを押すと、レス内にあるスレ番号を(最大12個)解釈し、リスト表示（Jumpリスト表示）するようにした。その中から番号を選択して "OK" を押すとそのメッセージにジャンプできるようにした。(OKは、Jog Pushでも可)
    • NNsi設定-4に、Backボタンを"Go"ボタンと等価にする設定を用意した
    • NNsi設定-4に、GoJumpメニューに「一覧へ戻る」アイテムを追加する設定を用意した
    • ハイレゾ描画モードをグラフィック描画モードに拡張し、としてCLIE以外でも使えるようにした。
    • 「スレ一覧を取得」メニューを「管理」カテゴリから「スレ」カテゴリに移動させる。
    • NNsi設定系の表記を一部見直した。
  • Version 0.92 Revision002 : 04/Dec/2002
    • NNsiからの起動をサポートするWebBrowserにBlazerを追加する。
    • スレタイ検索機能が「お気に入り」「取得済み全て」および「未読あり」では動作していなかった不具合を修正する。
    • スレタイ検索で文字列を入力せずに次検索/前検索を実施したとき、Fatal Errorが発生する不具合を修正する。
    • 「DBをVFSに保管」機能が動作する条件を緩和する。 (VFSが使用可能であれば実行可能にする)
  • Version 0.92 Revision001 : 30/Nov/2002

    • スレ一覧画面で、該当スレが１つもないとき、取得/参照/情報操作ができてしまう不具合を修正する。
    • 不要なメニューを隠していると、PalmOS V3.5以下(?)の機種でその隠れたメニュー部分を選択するとロックアップしてしまう不具合に対処（メニューを隠すことをやめました(苦笑)。）
    • メニューカテゴリを日本語に変更する。
    • NNsi終了時に、メモリを開放せず終了した部分を修正する。

    • 機種別の機能をメニューに割り当てる方法を変更する。（従来は、機種別の機能を後からメニューに追加する方法をとっていましたが、今バージョンから機種別の機能をメニュー表示時に隠すように変更してみました。(PowerJOG対策)）
